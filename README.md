## Requirements
- Python environment 
```
conda create -n audio_face
conda activate audio_face
```
- ffmpeg
```
sudo apt-get install ffmpeg
```
- python packages
```
pip install -r requirements.txt
```
- you may add opencv by conda.
```
conda install opencv
```
